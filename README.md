# AngularJS Node UI Components

## Prerequisites

### Git

- A good place to learn about setting up git is [here][git-github]
- Git [home][git-home] (download, documentation)

### Node.js

- Get [Node.js][node-download].

- install packages
- $ npm install

- grunt build
- $ grunt

- start Node server
- $ node server or dbl click server.bat
or
 $ grunt localhost
